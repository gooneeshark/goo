Gambling API Lab — Workshop

เป้าหมาย: ดึงดูดผู้เรียนด้วยการจำลอง "แฮ็ก" และสคริปต์ตัวอย่าง เพื่อให้เขามีแรงจูงใจมาเรียน JavaScript

  • รู้จัก fetch, headers, payload
  • ใช้ DevTools Simulator เพื่อดักและแก้คำขอ
  • รันสคริปต์ตัวอย่างจาก Library และแกะดูโค้ด

ใช้ปุ่มขวา/ซ้าย หรือปัด (มือถือ) เพื่อเลื่อนสไลด์

แผง DevTools Simulator

ฟีเจอร์สำคัญ:

  1. Intercept outgoing requests
  2. Edit headers/body
  3. Replay หรือ Send modified
  4. Forge requests (ทดลองปลอมคำขอ)

สาธิต: เปิด Intercept แล้วกด Spin ดูคำขอโผล่ในรายการ

Scripts Library — teaching by example

แนวทาง: ให้ผู้เรียนรันสคริปต์ตัวอย่างก่อน แล้วให้ดูโค้ด (Show code) เพื่อเรียนรู้แนวคิด

ตัวอย่างโค้ด (AutoSpin แบบย่อ):

(async ()=>{ for(let i=0;i<5;i++){ await apiFetch('/api/spin', {method:'POST', body: JSON.stringify({username:'player1',bet:10})}); await new Promise(r=>setTimeout(r,400)); }})();

ชี้: เปลี่ยน bet, ดู response, เพิ่ม retry — step-by-step

Workshop Tasks (สรุป)

  1. รัน AutoSpin แล้วดูผลใน simulated console
  2. ใช้ DevTools Simulator ดักคำขอแล้วแก้ header เป็นค่าอื่น (ลองดู 401 → แก้)
  3. แก้สคริปต์ให้เพิ่ม timeout/retry
  4. ดาวน์โหลด history เป็น CSV แล้วอธิบายข้อมูล
