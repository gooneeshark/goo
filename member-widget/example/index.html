Member Widget Demo

Embeddable authentication widget for any website

Quick Start

Add the Member Widget to your website with just a few lines of code:

<!-- Add widget container --> <div id="member-widget"></div> <!-- Load widget script --> <script src="member-widget.js"></script> <!-- Initialize widget --> <script> new MemberWidget({ containerId: 'member-widget', apiUrl: 'http://localhost:5173', theme: 'hacker', position: 'bottom-right' }); </script>

Configuration

Customize the widget appearance and behavior:

new MemberWidget({ containerId: 'member-widget', apiUrl: 'http://localhost:5173', theme: 'hacker', // 'hacker', 'dark', 'light' position: 'bottom-right', size: 'medium', // 'small', 'medium', 'large' collapsed: true, // Custom colors colors: { primary: '#00ff00', background: '#000000', text: '#ffffff' }, // Event callbacks onLogin: (user) => { console.log('User logged in:', user); }, onLogout: () => { console.log('User logged out'); } });
